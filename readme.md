# SAPOmanagement

## 1. Installation

- **node version 16**
- **express version 4.16**
- **Babel version 5**
- **Webpack version 7**
- **MongoDB for database**
```
npm install
```
## 2. For dev environment
```
npm run dev
```
### Router for frontend
```
https://localhost:3000
```
### Router for Backend API
```
https://localhost:3000/api/
```
